# my-aliases

> Custom aliases and functions configured in your dotfiles.
> Defined in ~/.aliases and ~/.functions, shared by both zsh and bash.

- Git shortcuts (status, log, diff, pull, push, commit, checkout, branch):

`gs  gl  gd  gp  gpu  gc  gco  gb`

- Docker shortcuts (docker, compose, ps, logs, exec, shell):

`d  dc  dps  dlog  dex  dsh {{container}}`

- Navigation (go up 1, 2, or 3 directories):

`..  ...  ....`

- Listing with eza (icons, long+git, tree, long):

`ls  ll  lt  l`

- Networking (public IP, local IP, listening ports):

`myip  localip  ports`

- Create dir and cd into it:

`mkcd {{dirname}}`

- Extract any archive format:

`extract {{file}}`

- Quick HTTP server in current directory:

`serve {{port}}`

- Terminal weather and timestamped file backup:

`weather {{city}}`
