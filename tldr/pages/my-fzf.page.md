# my-fzf

> Fuzzy finder shortcuts configured in your dotfiles.
> Uses fd for searching and bat for preview. Catppuccin Mocha theme.

- Fuzzy search command history:

`Ctrl+R`

- Fuzzy find a file and paste the path:

`Ctrl+T`

- Fuzzy jump to a directory:

`Alt+C`

- Fuzzy autocomplete any command (e.g. `cd **<Tab>`, `ssh **<Tab>`):

`**<Tab>`

- Fuzzy find and open a file in your editor:

`fe`

- Fuzzy kill a process:

`fkill`

- Fuzzy switch git branch:

`fbr`
