# my-git

> Git aliases and shortcuts configured in your dotfiles.
> Uses delta for syntax-highlighted diffs with Catppuccin Mocha theme.

- Quick status (short format with branch):

`gs`

- Pretty log (last 20 commits, graph):

`gl`

- Diff, diff staged changes:

`gd`

- Pull and push:

`gp`

- Commit with message:

`gc -m "{{message}}"`

- Checkout existing branch, create new branch:

`gco {{branch}}`

- Undo last commit (keep changes staged):

`git undo`

- Amend last commit without editing message:

`git amend`

- Fuzzy branch switcher (fzf-powered):

`fbr`
