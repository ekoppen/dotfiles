# my-tmux

> Tmux shortcuts configured in your dotfiles.
> Prefix key is Ctrl+Space (not the default Ctrl+B). Catppuccin Mocha theme.

- New named session:

`tn {{session-name}}`

- Attach to session:

`ta {{session-name}}`

- List sessions:

`tl`

- Kill session:

`tk {{session-name}}`

- Split pane horizontally / vertically:

`Ctrl+Space then -` / `Ctrl+Space then |`

- Navigate panes with vim keys:

`Ctrl+Space then h/j/k/l`

- Navigate panes with Alt+arrows (no prefix needed):

`Alt+Left/Right/Up/Down`

- Create new window:

`Ctrl+Space then c`

- Rename window:

`Ctrl+Space then ,`

- Enter copy mode (vim keys, scroll with j/k):

`Ctrl+Space then [`
